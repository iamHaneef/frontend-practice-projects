# Frontend Practice Projects

## 📌 Overview

A collection of frontend projects built to strengthen core web development skills through practical implementation.

These projects focus on user interface development, JavaScript fundamentals, DOM manipulation, event handling, and application logic.

---

## 🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

---

## 📂 Projects

| Project              | Description                                                                 |
| -------------------- | --------------------------------------------------------------------------- |
| Simple Calculator    | Performs basic arithmetic operations through an interactive user interface  |
| Number Guessing Game | Interactive game that challenges users to guess a randomly generated number |

---

## 🎯 Learning Focus

- HTML Structure
- CSS Styling
- Responsive Design
- DOM Manipulation
- Event Handling
- Application Logic
- User Interaction

---

## 🚀 Running the Projects

Clone the repository:

```bash
git clone <repository-url>
```

Open any project folder and launch the `index.html` file in your browser.
