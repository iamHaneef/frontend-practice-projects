# Number Guessing Game

## 📌 Overview

A simple browser-based game where the player guesses a randomly generated number within a defined range.  
The game provides hints to guide the player toward the correct answer.

---

## ⚡ Features

- Random number generation  
- User input handling  
- Hint system (higher / lower)  
- Simple interactive UI  

---

## 🛠️ Tech Stack

- HTML  
- CSS  
- JavaScript
